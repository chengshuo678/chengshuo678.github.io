普通

1.生成0和1的数组

np.zeros([3,3])

2.从现有数组中生成

np.array(one_array) # 深拷贝，全新一个实例

np.asarray(one_array) # 浅拷贝，指向原有

3.生成固定范围数组

np.linspace(0,10,5) # [0,10] 生成等间隔的5个item

np.arange(0,10,2) # [0,10] 生成以2为间隔生成
